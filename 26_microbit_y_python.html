2.6- Microbit y Python

Microbit y Python

La placa micro:bit puede programarse en Python para controlar sus componentes como su pantalla LED, botones y sensores. Se utiliza el entorno de programación en línea MakeCode de Microsoft, que ofrece soporte tanto para bloques como para texto (Python y JavaScript), permitiendo la conversión entre ellos. También existe el editor MicroPython para Micro:bit, una versión de Python adaptada a la placa que es más similar a la versión completa de Python 3. 

Os dejo un vídeo y un enlace por si queréis investigar como programar vuestra placa microbit con Python.
