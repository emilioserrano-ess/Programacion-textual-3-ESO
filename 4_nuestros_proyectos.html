4.- Nuestros proyectos

Programación educativa

Los conocimientos tecnológicos son cada vez más importantes dado que todo apunta a que van a ser imprescindibles para la mayoría de las profesiones del futuro.

Sin embargo, a pesar de los grandes beneficios que tiene desarrollar estos conocimientos a edades tempranas, en muchos países en desarrollo aún están lejos de empezar a impartirse en las aulas ya que, generalmente, los colegios de estos países no cuentan con recursos suficientes.

Una ONG nos ha pedido ayuda para llevar a cabo un proyecto que está empezando a desarrollar capaz de contribuir al aprendizaje de diversas temáticas técnicas, como matemáticas, física o química.

Los requisitos que nos piden son:

  • La finalidad del programa sea explicar o reforzar algún concepto educativo, preferiblemente técnico, con el fin de cumplir el ODS 4, Educación de calidad.
  • Dicho programa debe realizarse utilizando el lenguaje de texto Python.
  • Se debe documentar el código mediante comentarios para facilitar su comprensión.

Para ayudarnos a definir cómo va a ser la programación podemos escribir el algoritmo o pseudocódigo del programa completo y señalar el diagrama de flujo de la programación o hacer un esquema que represente su funcionamiento.

Finalmente, comprobamos el funcionamiento para detectar errores de programación y pensamos en posibles mejoras. Además, nos aseguramos que nuestra solución cumple con todos los requisitos pedidos.

Proyecto 1: Resolviendo ecuaciones de segundo grado con Python

Objetivo

Deberéis programar una aplicación en Python que resuelva cualquier ecuación cuadrática del tipo: ax² + bx + c = 0

Además de calcular las soluciones, el programa deberá indicar si la ecuación tiene 0, 1 o 2 soluciones reales.

Contenidos de programación a trabajar

  • Entrada de datos (input() para a, b y c).
  • Conversión de datos (de str a float).
  • Operaciones matemáticas (uso de potencias y raíces cuadradas con math.sqrt).
  • Condicionales (según el discriminante: Δ = b² - 4ac).
  • Funciones (opcional: crear una función que calcule las soluciones).
  • Gestión de errores (qué pasa si a = 0, no sería una ecuación de segundo grado).

Pasos del proyecto

1. Analizar el problema matemático:

La ecuación de segundo grado se aplica a ecuaciones del tipo ax² + bx + c = 0

Recordemos que la fórmula que resuelve este tipo de ecuaciones es:

\(x=\displaystyle \frac {-b \pm \sqrt{b^2-4ac}}{2a}\)

En realidad esta fórmula son dos fórmulas, una para el + y otra para el - delante de la raíz. Así que usaremos dos fórmulas en el programa:

\[x_1= \frac {-b + \sqrt{b^2-4ac}}{2a}\]

\[x_2= \frac {-b - \sqrt{b^2-4ac}}{2a}\]

Para determinar el número de soluciones de este tipos de ecuaciones hemos de calcular el discriminante: Δ = b² - 4ac.

  • Si Δ < 0 → no hay soluciones reales.
  • Si Δ = 0 → hay una única solución real.
  • Si Δ > 0 → hay dos soluciones reales.

2. Diseñar el flujo del programa

Pedir datos → calcular Δ → mostrar número de soluciones → imprimir las raíces si existen.

3. Implementar el programa en Python

Realizamos nuestro programa en Python siguiendo los pasos del diagrama de flujo.

4. Testamos y mejoramos nuestro programa

Resolvemos los posibles errores que dé nuestro programa y realizamos mejoras

  • Podemos crear una función llamada resolver_ecuacion(a, b, c) que devuelva las soluciones.
  • Repetir el programa en un bucle hasta que el usuario escriba “salir”.
  • Mostrar comentarios explicativos y mensajes motivadores que ayuden a la compresión del programa y del contenido matemático.

Proyecto 2: Simulador de caída libre

Objetivo

Deberéis crear un programa en Python que simule el movimiento de un objeto en caída libre, aplicando las fórmulas básicas del movimiento uniformemente acelerado (gravedad).

El programa deberá:

  • Pedir al usuario la altura inicial desde la que se deja caer el objeto.
  • Calcular el tiempo de caída hasta llegar al suelo.
  • Mostrar la posición del objeto en cada segundo.
  • Indicar si el objeto ya ha llegado al suelo.

Contenidos de programación a trabajar

  • Entrada de datos (input() para la altura).
  • Conversión de datos (de str a float).
  • Operaciones matemáticas (uso de potencias y raíces cuadradas con math.sqrt).
  • Condicionales (if para comprobar si el objeto ya ha llegado al suelo).
  • Bucles (while o for para simular cada segundo).
  • Funciones (una función que calcule la posición en un tiempo dado).

Pasos del proyecto

1. Analizar el programa físico

  • Fórmula de la posición en caída libre (sin velocidad inicial):

\(y(t)=h−\frac{1}{2}​gt^2\)

  • Tiempo total de caída:

\(t = \sqrt{\frac{2h}{g}}\)

  • Constante de gravedad: g = 9.8 m/s2.  

2. Diseñar el flujo del programa

Definir función posición y función tiempo → pedir altura → calcular tiempo caída y mostrarlo en pantalla → bucle para mostrar posición en cada segundo

3. Implementar el programa en Python

Realizamos nuestro programa en Python siguiendo los pasos del diagrama de flujo.

4. Testamos y mejoramos nuestro programa

Resolvemos los posibles errores que dé nuestro programa y realizamos mejoras

  • Permitir una velocidad inicial (lanzamiento hacia abajo o hacia arriba).
  • Representar la trayectoria gráficamente con matplotlib.
  • Repetir el programa en un bucle hasta que el usuario decida salir.
  • Mostrar una barra de progreso con texto (por ejemplo: ##### caída...)
